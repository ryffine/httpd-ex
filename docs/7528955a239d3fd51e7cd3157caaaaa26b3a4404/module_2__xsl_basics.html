Module 2: XSL basics

Come back and populate shortdesc

This module is designed to introduce participants to the eXtensible Stylesheet Language (XSL) set, with a focus on eXtensible Stylesheet Language Transformation (XSLT).

Participants will use the knowledge of XSLT to transform an XML file into DITA using common XSLT elements. Participants will take their knowledge of XSLT a step further in Module 5, when they overwrite the out-of-the-box DITA transformations. transformations.

After going through this module, participants will be able to:

  • Set up the environment to work with XSLT and describe the functionality of the panels and associated features in the XSLT view in Oxygen

  • Explain how XSLT steps through the stylesheet and the DITA file

  • Use XSLT/XPath to locate and transform specific DITA elements and attributes

  • Use XSLT/XPath to apply style to DITA elements
